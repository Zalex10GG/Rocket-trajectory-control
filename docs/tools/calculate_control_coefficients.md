# `tools/calculate_control_coefficients.py`

## Purpose

Estimates aerodynamic force derivatives for the `GenericSurface` control model. It reads rocket geometry from the TOML file and computes the incremental forces generated by fin deflections.

## Command

Run from the repository root:

```powershell
uv run py tools/calculate_control_coefficients.py
```

## Calculations

### 1. Lift Slope (Isolated Fin)
Using the Diederich lift slope for subsonic regimes:

$$C_{N_{\delta, single}} = \frac{2\pi \cdot s^2 / A_{ref}}{1 + \sqrt{1 + \left( \frac{\beta \cdot s^2}{A_{fin} \cdot \cos \Gamma_c} \right)^2}}$$

Where:
- $s$: Fin span.
- $A_{ref}$: Reference area ($\pi r_{body}^2$).
- $\beta = \sqrt{1 - M^2}$: Compressibility factor.
- $\Gamma_c$: Mid-chord sweep angle.

### 2. Interference and Scaling
Accounts for body-on-fin interaction ($K_{TB}$) and cruciform (+) configuration:

$$K_{TB} = 1 + \frac{r_{body}}{s + r_{body}}$$
$$C_{N_\delta} = C_{y_\delta} = 2 \cdot C_{N_{\delta, single}} \cdot K_{TB}$$

### 3. Induced Drag Factor
Based on the Aspect Ratio $AR$ and Oswald efficiency $e$:

$$k = \frac{A_{fin}}{2\pi \cdot s^2 \cdot e}$$
$$c_{D, induced} = k \cdot (c_L^2 + c_Q^2)$$

## Outputs

The script prints the calculated coefficients:
- `cN_delta_per_rad`: Pitch normal force derivative.
- `cy_delta_per_rad`: Yaw lateral force derivative.
- `k_drag_induced`: Induced drag factor.

These values should be updated in the `[control_actuation]` section of the rocket TOML file to improve simulation fidelity.
